# Fresh Signal — Demo Sites

Website mockups built by Nate Witt (Fresh Signal) for local Central Arkansas businesses.

Each page is a **demo mockup**, clearly labeled as such. Photos are sample stand-ins,
replaced with the owner's real images on launch. No reviews, ratings, or prices shown
are represented as real unless verified and marked.

All pages are `noindex` — they are shareable by link and invisible to search engines,
so they never compete with a business's real listing.
